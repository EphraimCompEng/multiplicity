    multiplied.core.utils.char.allchars(matrix: list[list[str]], *, hash=[]) → set[str][source]#

    Returns set of unique characters from a matrix. Ignores underscore characters and converts characters to uppercase

    >>> allchars([['A', 'B'], ['C', 'D']])
{'A', 'B', 'C', 'D'}

    hash: use checksum to assist in calculations

    multiplied.core.utils.char.chartff(ch: str) → Generator[str][source]#

    Generator to flip flop between upper and lowercase characters.

    >>> x = chartff('a')
>>> next(x)
'a'
>>> next(x)
'A'
>>> next(x)
'a'
